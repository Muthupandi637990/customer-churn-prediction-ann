# 📉 Customer Churn Prediction using Artificial Neural Network

## 📌 Project Overview
Predicts whether a customer will leave a service using Deep Learning (ANN).

## 📊 Dataset
Customer data including demographics, usage, and account information.

## 🧠 Techniques Used
- Data Preprocessing
- Feature Encoding
- ANN Model (Deep Learning)
- Hyperparameter Tuning

## 📈 Model Evaluation
- Accuracy
- Confusion Matrix
- Precision & Recall

## 🛠 Technologies
Python, TensorFlow, Keras, Pandas, NumPy

## ▶️ How to Run
pip install -r requirements.txt  
python train.py
